# 📱 校园交易系统 - 交易流程与安全说明

## 🎯 交易流程设计

### 1️⃣ 浏览商品阶段
```
用户进入商品市场 → 筛选/搜索商品 → 查看商品详情
```

**功能点:**
- 🔍 多维度筛选(分类/成色/价格/排序)
- 📱 网格/列表双视图
- 🏷️ 商品标签(可小刀/包邮/急出/支持验机等)
- 🎓 校内认证标识
- 👁️ 浏览量/咨询量统计

---

### 2️⃣ 评论沟通阶段
```
点击商品 → 切换到「评论留言」标签 → 发表留言 → 买卖双方沟通
```

**功能点:**
- 💬 发表评论留言
- 🔄 楼中楼回复功能
- 👤 卖家标识高亮显示
- 🎓 校内认证标识
- 🚫 举报违规评论
- ⏰ 时间戳显示

**典型对话场景:**
```
买家: "这个还在吗?成色怎么样?"
卖家: "在的!成色很好,无磕碰,可以当面验货"
买家: "好的,明天下午3点图书馆门口见面可以吗?"
卖家: "可以的,到时候见!记得带现金"
```

---

### 3️⃣ 获取联系方式阶段
```
点击「我想要 - 联系卖家」→ 阅读交易规则 → 确认购买意向 → 获取联系方式
```

**弹窗内容:**

#### ⚠️ 重要提醒
- 向卖家发送您的联系方式
- 向您显示卖家的联系方式
- 记录本次交易意向

#### ❌ 严禁行为(违者封号)
1. **线上转账支付**
2. **未见面先付款**
3. **发布虚假商品**
4. **恶意欺诈**

#### ✅ 正确流程
1. **线下约见面地点**
2. **当面验货**
3. **验货后付款**

#### 💡 温馨提示
建议选择校内公共场所交易,如:
- 图书馆门口
- 食堂广场
- 教学楼大厅
- 宿舍楼下

---

### 4️⃣ 线下交易阶段
```
约定时间地点 → 当面见面 → 验货检查 → 现金支付 → 交易完成
```

**安全要点:**
- ✅ 选择公共场所
- ✅ 白天交易更安全
- ✅ 可以叫上朋友陪同
- ✅ 当面验货后再付款
- ✅ 保留聊天记录

**验货清单:**
- [ ] 外观检查(磕碰/划痕)
- [ ] 功能测试(电子产品开机测试)
- [ ] 配件齐全(充电器/说明书/包装盒)
- [ ] 真伪鉴别(品牌商品注意仿品)

---

### 5️⃣ 交易完成阶段
```
交易完成 → 商品自动下架 → 双方可互相评价
```

**系统操作:**
- 🔒 商品状态变更为「已售出」
- 📊 统计数据更新
- ⭐ 开放评价功能
- 📝 交易记录存档

---

## 🛡️ 安全防护机制

### 1. 实名认证
- 🎓 校内邮箱认证
- 📱 手机号绑定
- 🆔 学生证验证

### 2. 信用评分系统
- ⭐ 卖家星级评分(1-5星)
- 📊 成交记录统计
- 💯 好评率显示
- 🏆 优质卖家标识

### 3. 违规处理
```
发现违规 → 用户举报 → 管理员审核 → 处罚措施
```

**处罚等级:**
- ⚠️ 警告: 首次轻微违规
- 🚫 限制发布: 7天内禁止发布商品
- ❌ 封号: 严重违规直接封号

**违规行为:**
1. 线上转账诈骗
2. 发布违禁商品
3. 虚假商品信息
4. 恶意抬价/压价
5. 辱骂/骚扰用户

### 4. 交易记录
- 📝 所有评论留言永久保存
- 💬 联系方式获取记录
- ⏰ 完整时间轴追溯
- 🔍 管理员可查看调取

## 📋 用户操作指南

### 买家操作流程

#### 步骤1: 找到心仪商品
1. 进入商品市场
2. 使用筛选/搜索功能
3. 点击商品查看详情

#### 步骤2: 评论留言沟通
1. 切换到「评论留言」标签
2. 发表留言询问商品细节
3. 等待卖家回复
4. 确认商品状况

#### 步骤3: 获取联系方式
1. 点击「我想要 - 联系卖家」
2. 仔细阅读交易规则
3. 点击「确认购买意向」
4. 获得卖家联系方式

#### 步骤4: 约定交易
1. 通过微信/QQ联系卖家
2. 约定见面时间和地点
3. 确认商品和价格

#### 步骤5: 当面交易
1. 准时到达约定地点
2. 当面验货
3. 确认无误后付款
4. 交易完成

---

### 卖家操作流程

#### 步骤1: 发布商品
1. 点击「我要卖」
2. 填写商品信息:
   - 商品名称
   - 分类选择
   - 价格设置
   - 成色标注
   - 详细描述
   - 上传图片(最多5张)
   - 联系方式
3. 提交发布

#### 步骤2: 回复买家咨询
1. 关注商品评论区
2. 及时回复买家问题
3. 详细介绍商品状况
4. 约定见面时间地点

#### 步骤3: 完成交易
1. 准时赴约
2. 展示商品
3. 收取款项
4. 交付商品

#### 步骤4: 后续处理
1. 商品自动下架
2. 查看买家评价
3. 累积信用分

## 💡 常见问题 FAQ

### Q1: 为什么不能线上支付?
**A:** 为了保护双方安全:
- 避免转账诈骗
- 确保商品真实性
- 当面验货更放心
- 减少纠纷发生

### Q2: 如果对方爽约怎么办?
**A:** 
- 保留聊天记录
- 向平台举报
- 平台会降低其信用分
- 严重者封号处理

### Q3: 商品有问题可以退吗?
**A:**
- 建议当面验货
- 发现问题直接拒绝交易
- 交易前沟通清楚
- 二手商品一般不退换

### Q4: 如何判断卖家是否可信?
**A:** 看以下指标:
- ⭐ 卖家星级评分
- 🎓 是否校内认证
- 📊 历史成交记录
- 💬 评论区回复质量

### Q5: 遇到诈骗怎么办?
**A:**
1. 立即停止交易
2. 保留证据(聊天记录/转账记录)
3. 向平台举报
4. 必要时报警处理

### Q6: 可以讨价还价吗?
**A:**
- 可以在评论区礼貌沟通
- 标注「可小刀」的商品可以小幅议价
- 尊重卖家决定
- 恶意砍价会被举报
